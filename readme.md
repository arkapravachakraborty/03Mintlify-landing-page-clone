# How to run this on your local machines
Since this project uses pure HTML and CSS , there are no dependencies or build steps required. Just clone the repository and change the details according to your preferences.

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/arkapravachakraborty/03Mintlify-landing-page-clone
    ```
2.  **Navigate to Folder:**
    ```bash
    cd 03Mintlify-landing-page-clone
    ```
3.  **Launch the Page:**
    * **Option A:** Simply double-click `index.html` to open it in your default browser.
    * **Option B:** Use the **Live Server** extension in VS Code for real-time updates while editing.
